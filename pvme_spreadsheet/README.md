# Development

## Install

```commandline
cd pvme-settings\pvme_spreadsheet
```
```toml
# .env
GS_URL = "<GS_URL>"
GS_PRIVATE_KEY = "<GS_PRIVATE_KEY>"
GS_CLIENT_EMAIL = "<GS_CLIENT_EMAIL>"
GS_TOKEN_URI = "<GS_TOKEN_URI>"
```
```commandline
virtualenv venv
venv\Scripts\activate
pip install pip-tools
pip install python-dotenv
```
